Detailed Description

远程渲染器代理类，实现了代理模式。

主要功能：

  • 在调用实际渲染方法前后添加日志记录
  • 控制对真实渲染器的访问
  • 可以添加额外的网络通信逻辑

典型用法： 



Renderer realRenderer = new SwingRenderer();

Renderer proxy = new RemoteRendererProxy(realRenderer);

proxy.drawCircle(100,100,50); // 通过代理调用
