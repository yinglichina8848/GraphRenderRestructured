Detailed Description

渲染器工厂，支持动态扩展渲染器实现。

扩展机制： 

// 注册自定义渲染器
RendererFactory.register("custom", CustomRenderer::new);

// 创建实例
Renderer renderer = RendererFactory.create("custom");

内置渲染器：

  • swing - SwingRenderer
  • svg - SVGRenderer
  • test - TestRenderer
  • legacy - LegacyRendererAdapter
