Detailed Description

Command接口定义了命令模式的基本操作。 所有具体命令类必须实现execute()和undo()方法。

命令模式允许将请求封装为对象，从而支持：

  • 参数化客户端请求
  • 请求排队或记录
  • 支持可撤销操作

典型用法： 

Command cmd = new AddShapeCommand(shapes, new Circle(10,10,5));
cmd.execute(); // 执行命令
cmd.undo();    // 撤销命令
