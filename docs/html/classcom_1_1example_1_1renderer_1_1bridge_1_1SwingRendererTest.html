Detailed Description

SwingRenderer的单元测试类。

验证Swing渲染器的以下功能：

  • 正确委托给Graphics2D对象
  • 各种图形的渲染调用
  • 极端坐标处理
  • 退化图形情况

使用Mockito模拟Graphics2D对象进行验证。
