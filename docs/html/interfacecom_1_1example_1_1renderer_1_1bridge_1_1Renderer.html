Detailed Description

Renderer接口定义了图形渲染的基本操作。 作为桥接模式中的实现部分(Implementor)，它抽象了不同渲染方式的具体实现。

实现类需要提供以下图形的绘制能力：

  • 圆形 - 通过drawCircle方法
  • 矩形 - 通过drawRectangle方法
  • 椭圆 - 通过drawEllipse方法
  • 三角形 - 通过drawTriangle方法

典型用法： 

Renderer renderer = new SwingRenderer();


renderer.drawCircle(100, 100, 50); // 绘制圆形
