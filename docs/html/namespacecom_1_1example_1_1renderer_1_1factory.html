Detailed Description

Circle 类实现了圆形图形的具体表示和操作。

作为 Shape 接口的实现类，它提供了圆形的核心功能：

  • 存储圆心坐标(x,y)和半径(radius)
  • 通过 render() 方法使用指定渲染器绘制圆形
  • 支持通过 move() 方法移动圆形位置
  • 支持通过 accept() 方法接受访问者操作

构造器参数验证：

  • 半径必须为正数，否则抛出 IllegalArgumentException

典型用法： 

Circle circle = new Circle(100, 100, 50); // 创建圆形


circle.render(renderer); // 绘制圆形


circle.move(10, 10); // 移动圆形

表示椭圆图形的实现类。

椭圆由中心坐标(x,y)和宽度(width)、高度(height)定义。 该类实现了Shape接口，支持基本的渲染、移动和访问者模式操作。

遵循的设计模式：

  • 桥接模式 - 通过Renderer接口实现绘制逻辑
  • 访问者模式 - 通过accept方法支持扩展操作
  • 命令模式 - move方法支持位置变更

矩形图形实现类。

表示一个矩形图形，包含以下属性：

  • x,y: 左上角坐标
  • width: 宽度(必须>0)
  • height: 高度(必须>0)

实现了Shape接口的所有方法，包括：
