Detailed Description

移动图形命令实现类。

封装了移动图形的操作，支持执行和撤销。

典型用法： 

Shape shape = new Circle(10, 10, 5);


Command cmd = new MoveShapeCommand(shape, 20, 30);


cmd.execute(); // 移动图形


cmd.undo();   // 撤销移动
